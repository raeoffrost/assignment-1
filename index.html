Hyperlinks

As you can see to the left (or above if you are on a tiny screen) there is an image of the YouTube logo. This image as well as the sentence you are currently reading are hyperlinks! Clicking on either will take you to YouTube and thanks to the handy dandy "target" attribute they open in another tab.

The 'rel' Attribute

While I have previous knowledge of how to create hyperlinks I did take the opportunity to learn something new!
The "rel" attribute helps in specifying the relationship between the link and linked item. This is particularly useful for search engines and analytics. I have added the "noreferrer" value to my rel attribute. This makes the website that referenced the link unknown.
More can be learned about referers here.
